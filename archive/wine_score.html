Text-based Wine Quality Prediction

Project Date

Dec 17, 2019

Category

Spark, Scala, Machine Learning,
NLP, Data Wrangling, Regression

  Report   View Code

Background

During my graduate studies, I had the opportunity to implement a machine learning model within a DataBricks Spark cluster environment. The task involved developing a model capable which takes English text from reviews as input and predicts quality scores associated with different wines. Data was obtained from Kaggle, where it was originally scraped from the WineEnthusiast website.

The following regression models were implemented in Spark:

  • Linear Regression (LR)
  • Decision Tree
  • Random Forest
  • Gradient Boosted Trees

Text featurization techniques investigated included:

  • Stemming
  • TF-IDF
  • Word2Vec

Results

Running a spot check of the regression algorithms, LR outperformed the other tree-based models when either TF-IDF or Word2Vec features were used. I suspected that the tree-based models were unable to model linear relations effectively.

Upon further investigation and hyperparameter tuning, the best found model was a LR model with L2 regularization utilizing TF-IDF as feature inputs. The test set RMSE was 1.76 with an R2 of 0.68. I think the model could have been further improved by incorporating the other feature information provided (country, price, etc.) in addition to text data.
