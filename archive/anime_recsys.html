Building an Anime Recommendation System

Project Date

Jun 2, 2020

Category

Python, Recommender Systems, Data Wrangling,
Matrix Factorization, Model Evaluation

nbviewer View Code

Background

As a personal project, I decided to implement various kinds of recommender systems in Python from scratch since they have become increasingly popular at recommending relevant content to users across different platforms. Given my interest in anime, I decided to recommend anime content (TV series, movies, etc.) to users based on explicit rating data collected from MyAnimeList.

In particular, I implement and evaluate the following approaches:

  • Popularity Recommender (baseline)
  • Content-based Filtering
  • Collaborative Filtering (User-based, Item-based, Matrix Factorization)

Results

To evaluate how good a set of recommendations were, I used normalized discounted cumulative gain (NDCG@K) for the 'top K' items as a ranking metric (K=10).

After model evaluation on test data, it was apparent that the Matrix Factorization (MF) model provided the best user recommendations. The top ten MF recommendations for a user are shown below. In second was the baseline popularity recommender. The other models yielded poor performance due to their sensitivity to sparse rating data. Dealing with sparse rating data is a common challenge when there are many products for users to interact with on a platform.
